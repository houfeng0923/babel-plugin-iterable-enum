# babel-plugin-iterable-enum

Make typescript enum iterable.

## Install

Using npm:

```sh
npm install --save-dev babel-plugin-iterable-enum
```

or using yarn:

```sh
yarn add babel-plugin-iterable-enum --dev
```

## Usage

`.babelrc`

```json
{
  "plugins": ["iterable-enum", "@babel/transform-typescript"]
}
```
